# kevinBacon
Computes the shortest path from any actor to Kevin Bacon

Creates a graph structure to store actor information.
The vertices store the actor's name and the edges store the movie's title. This graph is used to
compute the degrees of separation between the actor Kevin Bacon to all the other actors. User can
repeatedly enter an actor's name and print a shortest path from that actor to Kevin Bacon that
lists the movies that connect them.
